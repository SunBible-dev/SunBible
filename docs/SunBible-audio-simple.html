SunBible Audio

Hi, it is me NCR beta testing how to include audio with SunBible!
You can click the audio logo to toggle this sidebar.
The audio will continue playing as you follow along or navigate to different pages.

For more info and to leave feedback please join the SunBible discord server:
https://discord.gg/qMt7NhJBVM
There is also a link to join the discord server in the context menu that will open in your browser.
(Everything you need is in the context menu.
And if you have any suggestions for more items to add to the context menu or just any suggestions or features you would like to see added to SunBible. Just join the SunBible discord server.)
